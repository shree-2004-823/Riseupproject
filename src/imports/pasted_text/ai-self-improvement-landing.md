Design a modern, premium, AI-powered self-improvement web application landing page and UI system.

The product is an AI-driven platform that helps users improve their physical, mental, and emotional health through habit tracking, mood monitoring, journaling, and an intelligent AI coach. It also helps users quit bad habits like smoking and drinking, while providing daily motivation, structured plans, and progress analytics.

Overall Design Style

Create a visually stunning, modern, and emotionally engaging UI with:

Dark premium theme (primary background: deep charcoal or near-black)
Gradient accents (electric blue, violet, emerald green, subtle orange highlights)
Soft glow effects for primary elements
Rounded 2xl cards and smooth shadows
Minimal glassmorphism (only for key elements like navbar and chatbot)
Clean spacing and modern typography (large bold headings, soft body text)
Calm yet energetic aesthetic (balance between wellness app and tech product)
